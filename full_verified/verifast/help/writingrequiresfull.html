VeriFast Help > Error messages

Error message

Writing to a field requires full field permission.

Context

When assigning to a field.

Cause

The symbolic heap contains a field chunk for the assigned field, but the fraction of that chunk is smaller than 1.

Notes

  • VeriFast automatically merges two field chunks if it follows from the path condition that their target is equal.

Suggested solutions

Increase the fraction of the field chunk to 1. Suggested approaches:

  • If the code preceding the assertion leaks the heap chunk, remove the leak command.
  • Use one of the solutions of No matching heap chunks.
