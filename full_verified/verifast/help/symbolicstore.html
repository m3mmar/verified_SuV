Term

Symbolic store

Definition

A function mapping local variables to their symbolic values. The symbolic value of a variable is a first-order term.

Shown in the Locals pane in the top right corner of the VeriFast window.

Notes

  • To show the symbolic store at a given symbolic execution step, select the step in the Steps pane.
  • An entry "x=t" in the symbolic store indicates that the local variable x has symbolic value t.
  • The initial value of a variable or function parameter "x" is the first-order variable "x".
  • The local variable "currentThread" is a ghost variable that represents the id of the current thread.
  • The path condition, the symbolic store, and the symbolic heap together constitute the symbolic state of the symbolic execution.
